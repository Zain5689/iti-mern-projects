one-time compilation

Watch Mode => Terminal must be running in the background
Terminal: sass --watch source_folder:destination_folder
Press Ctrl-C to stop

In Development Phase (Readable CSS)
Terminal: sass --watch source_folder:destination_folder --style=expanded


In Production Phase (Optimized CSS)
Terminal: sass --watch source_folder:destination_folder --style=compressed


The 7-1 Pattern (Folder Organization)
scss/
  |________ abstracts/
  |             |________ _variables.scss
  |             |________ _functions.scss
  |             |________ _mixins.scss
  |             |________ _placeholders.scss
  |             
  |________ base/
  |           |________ _reset.scss
  |           |________ _typography.scss
  |           |________ _styles.scss
  |
  |________ components/ # Buttons/Cards/Models
  |________ layout/ # Header/Footer/Grid System
  |________ pages/ # Page-specific styles
  |________ themes/ # Color themes, dark mode
  |________ vendors/ # Third-Party Libraries (Bootstrap, plugins)
  |________ main.scss # Single entry point
