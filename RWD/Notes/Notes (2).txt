Physical Pixel: Real Hardware dot screen

CSS/Logical Pixel: Virtual unit used by CSS
Created by Browser
Scales Automatically


DPR: Device Pixel Ratio = Physical Pixel / Logical Pixel
Example:
DPR = 3
1 CSS Pixel = 3 Physical


Physical width = 1170 pixels
CSS width = 390 pixels


Viewport: Area where the browser lays out the page

Layout Viewport: used for CSS layout Calculations
Width used by media queries
Default value (Historically): 980px

Visual Viewport: What the user actually sees
Changes when you zoom
Smaller than Layout Viewport when Zoomed
